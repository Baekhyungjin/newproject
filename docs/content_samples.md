# 콘텐츠 샘플

아래 내용은 “성경 검색 + 관점 제안 + 매일 묵상” 시스템의 초기 샘플입니다.

## 1) 샘플 CLI 출력 화면

```
$ bible search "요한복음 3:16"

────────────────────────────────────────────────────────────
📖 본문
요한복음 3:16
하나님이 세상을 이처럼 사랑하사 독생자를 주셨으니 이는 그를 믿는 자마다
멸망하지 않고 영생을 얻게 하려 하심이라

────────────────────────────────────────────────────────────
🧭 관점 제안
1) 역사적 관점
   - 당시 유대-헬라 문화권에서 “사랑”과 “희생”의 의미가 어떻게 이해되었는지
     살펴볼 수 있습니다.
2) 문학적 관점
   - 요한복음 전체의 “빛/어둠, 생명/죽음” 대비 흐름 속에서 3:16의 위치를
     확인해보세요.
3) 목회적 관점
   - 오늘의 삶에서 “하나님의 사랑을 신뢰하는 믿음”이 어떤 행동으로
     드러날 수 있을지 생각해봅시다.

────────────────────────────────────────────────────────────
📝 오늘의 묵상
요약: 하나님의 사랑은 추상적인 감정이 아니라, 구체적인 선물로 드러납니다.
적용 질문:
 - 오늘 나는 하나님의 사랑을 어떻게 경험했는가?
 - 그 사랑에 대한 나의 반응은 무엇인가?
기도:
 - 사랑을 신뢰하는 믿음을 주시고, 그 사랑에 합당하게 살게 하소서.

Tip: `bible today`로 오늘의 묵상을 바로 확인할 수 있습니다.
```

```

본문: 시편 23:1-2

적용 질문:

 - 하나님께 맡겨야 할 걱정은 무엇인가요?
기도:
 - 주님, 제 마음을 고요하게 하시고 당신의 인도하심을 신뢰하게 하소서.
```

```
$ bible meditate "로마서 8:28"

묵상 안내:
1. 오늘 나의 삶에서 “선”을 신뢰하기 어려운 일은 무엇인가요?
2. 그 상황 속에서 하나님께서 일하실 여지를 어떻게 볼 수 있나요?
```

## 2) 초기 데이터 입력 포맷 (JSON/CSV)

### 2-1) 성경 본문 (JSON)

```json
{
  "book": "요한복음",
  "chapter": 3,
  "verse": 16,
  "text": "하나님이 세상을 이처럼 사랑하사 독생자를 주셨으니 이는 그를 믿는 자마다 멸망하지 않고 영생을 얻게 하려 하심이라",
  "translation": "개역개정"
}
```

### 2-2) 성경 본문 (CSV)

```csv
book,chapter,verse,text,translation
요한복음,3,16,"하나님이 세상을 이처럼 사랑하사 독생자를 주셨으니 이는 그를 믿는 자마다 멸망하지 않고 영생을 얻게 하려 하심이라",개역개정
```

## 3) 관점 템플릿 예시 (10개)

1. **역사적 관점**
   - 당시 문화/사회 배경에서 본문의 의미를 조명합니다.
2. **문학적 관점**
   - 본문 구조, 반복되는 키워드, 전후 문맥을 분석합니다.
3. **신학적 관점**
   - 하나님의 속성, 구속사 흐름, 교리적 포인트를 강조합니다.
4. **목회적 관점**
   - 현재의 삶에 적용할 수 있는 위로/도전 포인트를 제시합니다.
5. **윤리적 관점**
   - 행동과 선택의 기준, 공동체적 책임을 강조합니다.
6. **경건적 관점**
   - 기도와 예배, 내면의 변화에 초점을 둡니다.
7. **선교적 관점**
   - 세상과 이웃을 향한 복음의 확장과 증언을 강조합니다.
8. **공동체적 관점**
   - 교회/가정/소그룹 차원에서 적용할 수 있는 메시지를 제공합니다.
9. **상징적 관점**
   - 비유와 상징이 가진 의미를 조명합니다.
10. **현대적 관점**
    - 현대 사회의 문제와 연결되는 적용점을 제시합니다.

## 4) 묵상 예시 7일치

### Day 1 (시편 23:1-2)
**요약:** 하나님은 우리의 결핍을 아시고 필요한 곳으로 인도하십니다.  
**적용 질문:**  
- 지금 내 마음이 쉼을 필요로 하는 영역은 어디인가요?  
- 하나님께 맡겨야 할 걱정은 무엇인가요?  
**기도:** 주님, 제 마음을 고요하게 하시고 당신의 인도하심을 신뢰하게 하소서.

### Day 2 (잠언 3:5-6)
**요약:** 내 지혜보다 하나님의 인도를 신뢰할 때 길이 바르게 됩니다.  
**적용 질문:**  
- 내가 스스로 결정하려는 영역은 무엇인가요?  
- 하나님께 맡길 수 있는 작은 순종은 무엇인가요?  
**기도:** 주님, 제 길을 당신께 맡기고 순종하게 하소서.

### Day 3 (마태복음 5:14-16)
**요약:** 우리는 세상의 빛으로 부름받아 선한 행실로 하나님을 드러냅니다.  
**적용 질문:**  
- 오늘 내가 비출 수 있는 작은 빛은 무엇인가요?  
- 내 삶에서 누군가를 격려할 방법은 무엇인가요?  
**기도:** 주님, 제 삶을 통해 당신의 빛이 드러나게 하소서.

### Day 4 (로마서 8:28)
**요약:** 하나님은 모든 일을 선으로 이루시는 분입니다.  
**적용 질문:**  
- 지금 나에게 어려운 일은 무엇인가요?  
- 그 상황 속에서도 하나님을 신뢰할 수 있을까요?  
**기도:** 주님, 제 삶의 모든 순간에 당신의 선하심을 신뢰하게 하소서.

### Day 5 (요한복음 15:5)
**요약:** 우리는 하나님과의 연결 속에서 열매를 맺습니다.  
**적용 질문:**  
- 내 영적 생명력이 약해지는 순간은 언제인가요?  
- 하나님께 붙어 있기 위해 무엇을 실천할 수 있나요?  
**기도:** 주님, 당신 안에 거하여 열매 맺게 하소서.

### Day 6 (빌립보서 4:6-7)
**요약:** 염려를 내려놓고 기도할 때 하나님의 평안이 마음을 지킵니다.  
**적용 질문:**  
- 지금 나의 가장 큰 염려는 무엇인가요?  
- 그 염려를 기도로 바꾸기 위해 할 수 있는 일은 무엇인가요?  
**기도:** 주님, 제 마음에 당신의 평안을 채워 주소서.

### Day 7 (갈라디아서 6:9)
**요약:** 선을 행함에 낙심하지 말고 때가 되면 열매를 맺습니다.  
**적용 질문:**  
- 내가 포기하고 싶었던 선한 일은 무엇인가요?  
- 오늘 다시 시작할 수 있는 작은 선은 무엇인가요?  
**기도:** 주님, 선을 행함에 낙심하지 않게 하시고 끝까지 인내하게 하소서.
